WORLD DISASTER COMPILATION
[work]

inspired by worldisastercompilation.1 by Notinbed, this interactive visual project translates the music’s raw, destructive energy into image. it begins in a calm, almost meditative landscape before interaction triggers a gradual collapse, where real disaster footage, effects, and structures accumulate. the experience unfolds from stillness to total chaos, echoing both human responsibility and uncontrollable forces behind global catastrophes.



software: Visual Studio Code
